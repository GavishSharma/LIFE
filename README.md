# LIFE-Localized-Intelligent-Fleet-Ecosystem
LIFE (Localized Intelligent Fleet Ecosystem) is a small-scale swarm robotics project where multiple ESP32-based mobile robots coordinate intelligently under a centralized overhead vision system. 
A mounted high-resolution camera tracks each robot using ArUco markers, providing precise global position and orientation. 
The Raspberry Pi processes this vision data, manages communication, and assigns tasks to robots over Wi-Fi, while each robot handles its own movement and local collision avoidance. 
The system enables formation control, cooperative movement, environment awareness, and scalable swarm behavior suitable for research, education, and advanced robotics experimentation.
